Project Overview

Problem

Law enforcement agencies need data-driven insights to understand crime patterns and allocate resources effectively across Los Angeles.

Approach

Analyzed 901,625 crime incidents using Python for data cleaning and statistical analysis, then created interactive Tableau dashboards for visualization.

Solution

Developed comprehensive visualizations revealing temporal patterns, geographic hotspots, and demographic distributions across 21 LAPD divisions.

Impact

Identified peak crime hours (12 PM - 8 PM) and high-risk areas, enabling data-driven resource allocation and patrol scheduling decisions.

Key Insights

  • Temporal Patterns: Crime incidents peak between 12 PM and 8 PM, with notable variations by crime type and day of week
  • Geographic Distribution: Central and 77th Street divisions show highest incident rates, requiring focused intervention strategies
  • Demographic Analysis: Victim age distribution reveals vulnerability patterns across different crime categories
  • Crime Type Trends: Property crimes dominate incident reports, with distinct patterns varying significantly by division
